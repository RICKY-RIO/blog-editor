# Blog Editor

A full-stack blog editor with auto-save draft functionality.

## Features
- React frontend for editing and auto-saving blog drafts
- Express.js backend API
- MongoDB Atlas for data storage

## How to Run

### Backend
1. `cd backend`
2. `npm install`
3. `node server.js`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

## Technologies Used
- React
- Express.js
- MongoDB Atlas
- Node.js
